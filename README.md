# Phylogeny-of-Methanogenic-Genes-in-Anaerobic-Digestion
This repository contains the analysis code and processed data used in the study: Phylogenetic assembly of methanogenesis regulates methane yield in food-waste anaerobic digestion (Submitted)
## Repository Structure

### Statistical_analysis_code/
R scripts used for statistical analyses and figure generation.

### AD_Methane_Data/
Processed datasets used as inputs for statistical analyses.

### REMIRGE_modified/
Modified REMIRGE implementation used for reconstructing gene sequence clusters (GSCs).

## Data Availability

All data included here are processed datasets sufficient to reproduce the statistical analyses presented in the manuscript.
